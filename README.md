#  To-Do Web App

A responsive and interactive **To-Do Web Application** built using **HTML, CSS, and JavaScript** as part of my **SkillCraft Technology Web Development Internship (Task 4)**.

The application allows users to efficiently manage daily tasks with features like adding, editing, deleting, completing tasks, assigning priorities, searching, filtering, sorting, and persistent storage using Local Storage.

---

## Features

-  Add new tasks
-  Edit existing tasks
-  Delete tasks
-  Mark tasks as completed
-  Set due date & time
-  Assign task priority (High, Medium, Low)
-  Search tasks instantly
-  Filter tasks (All / Pending / Completed)
-  Sort tasks by newest, oldest, or priority
-  Task statistics (Total, Completed, Pending)
-  Local Storage support (tasks remain after page refresh)
-  Fully responsive design
